CODSOFT - Artificial Intelligence Internship

This repository contains the practical tasks and projects completed during my Artificial Intelligence internship with CodSoft. These projects bridge the gap between core programming logic—including my ongoing Bachelor of Computer Applications coursework—and functional AI implementations. 

The implementations rely heavily on structured control flow, algorithmic problem-solving, and database logic (applying concepts from C, Java, and SQL) to build efficient and scalable AI models.

📌 Projects Included
Task 1: Chatbot with Rule-Based Responses
A natural language processing tool that responds to user inputs based on predefined rules.  Concepts Used: Control flow (if-else statements), pattern matching, and conversation logic.  Description: This chatbot identifies specific keywords and query structures from the user to provide appropriate, hard-coded responses, establishing a baseline understanding of conversational AI.

Task 2: Tic-Tac-Toe AI
An unbeatable AI agent designed to play the classic game of Tic-Tac-Toe against a human player.  Concepts Used: Game theory, search algorithms, and discrete mathematics.Description: This project implements the Minimax algorithm to evaluate all possible future moves on the board, ensuring the AI never loses a match.  

Task 4: Recommendation System
A suggestion engine that recommends items to users based on their preferences.  Concepts Used: Data structures, filtering algorithms, and user-item mapping.Description: Utilizing techniques like collaborative filtering or content-based filtering, this system analyzes user data to accurately predict and suggest relevant items, such as movies or books.
